- Recharts: 차트/시각화
- TanStack Query (React Query): 서버 상태 관리 > API 데이터 페칭/캐싱

- Radix UI + Tailwind CSS: UI 컴포넌트
- lucide-react: Icon set
- clsx + tailwind-merge: Style Utils > 조건부 클래스 병합
